Frequently asked

Is the billing real?

No this project is a course assignment. "Upgrading" instantly unlocks features for the current account without any payment gateway. The implementation models how a real SaaS would gate features behind a plan.

Can I switch plans anytime?

Yes. Each Subscription row has a plan_type field; the subscribe view updates it directly. Existing data is never deleted when you change plans.

What if I go over a limit?

You won't lose data, but you can't add new projects or papers beyond the cap. The relevant view redirects you here with a friendly upgrade prompt.

Which features need a paid plan?

All AI features (paper summary, citation suggestions, project synthesis, summary refinement), the four non-APA citation styles, and visual comparison tables. Everything else is on Free.
